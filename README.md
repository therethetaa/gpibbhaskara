# GPIB Bhaskara Sby

## Description

GPIB Bhaskara Sby Front-end.

## Required Extension

```bash
- Prettier
- Tailwind CSS Intellisense
```

## Dependencies Installation

```bash
$ npm i
```

## Running the app

```bash
$ npm run dev

```

## Changing prettier CRLF to LF globally

```bash
$ prettier --write --end-of-line lf "**/*.{js,jsx,ts,tsx,css,scss,json,md}"
```
